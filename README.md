# Java To-Do List App

A console-based To-Do List application built using **Java** that helps users manage their daily tasks efficiently.
The application allows users to add, edit, delete, search, and organize tasks with priorities and categories.

---

## Features

* Add new tasks
* Edit task descriptions
* Delete tasks
* Mark tasks as completed
* Task priority levels (LOW / MEDIUM / HIGH)
* Task categories (WORK / STUDY / PERSONAL / HEALTH)
* Search tasks by keyword
* Filter tasks by category
* View task statistics
* Persistent task storage using file handling

---

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* ArrayList
* File Handling
* Java Date & Time API

## How to Run

1. Clone the repository

```
git clone https://github.com/MOHANX-0001/Java-ToDo-List-App.git
```

2. Compile the program

```
javac *.java
```

3. Run the application

```
java Main
```
